Why I Chose Octopress & Github

What's Octopress

Octopress is "A blogging framework for hackers". It's based on Jekyll Which is ruby statics web generator. Octopress is hackers so you should be familiar with shell and git commands or at least have the courage try.

Why github

Github is offering service for hosting your static pages for free Github Pages powered by Jekyll under the hod. This help you to deploy your Octopress blog easily. And git offer more flexible ways to manage your blog content.

You didn't tell me why Octopress

I chosen Octopress for couple of reasons:

  • markdown: I love to use it when ever it's possible. At Github, Stackoverflow or writing emails.

  • sharing code: With Octopress sharing snippets of code is easier then ever. Either with markdown or with gist tag

    import something
print 'markdown'

  • Git: deploying of Octopress help me to learn more on git (All I knew before was clone, pull and push). And the process of posting help me understand branches and practice them.
